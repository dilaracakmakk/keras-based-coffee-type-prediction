Keras-Based Coffee Type Prediction

This is a machine learning application built with Python and Keras to predict coffee types based on predefined flavor/aroma profiles.  
The project includes a trained model, GUI interface, and user profile management system.



Features

Predicts coffee type based on user preferences
Uses a trained `.keras` model with Keras
GUI built using `Tkinter`
Supports profile saving and data augmentation
Data is stored in `.json` and `.csv` formats

---

Model Details

Framework: TensorFlow / Keras
Training Script: `model_train.py`
Pretrained Model: `coffee_model.keras`
